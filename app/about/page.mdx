# About

## Peter's Archive에 오신 것을 환영합니다

PARA 방법론과 제텔카스텐을 결합하여 체계적으로 정보를 관리합니다.

---

## 📖 블로그 구조

이 블로그는 **PARA + Zettelkasten** 방식으로 구성되어 있습니다.

### 📊 Areas (영역)
**지속적으로 관리하는 분야**

끝이 없고 계속 관심을 가지는 영역들입니다.

**예시:**
- 투자 (미국 주식, 암호화폐, 부동산)
- 개발 (Python, 웹 개발, 알고리즘)
- 비즈니스 (마케팅, 제품 기획)

---

### 🚀 Projects (프로젝트)
**명확한 목표가 있는 단기 작업**

시작과 끝이 있으며, 완료되면 Archives로 이동합니다.

**예시:**
- Jekyll 블로그 만들기 (목표: 2026.03.30)
- Python 자동매매 봇 개발
- React 투두리스트 앱

---

### 💡 Notes (노트)
**하나의 아이디어만 담은 원자적 노트**

제텔카스텐(Zettelkasten) 방식으로 관리합니다.

**특징:**
- 노트 간 연결을 통한 지식 네트워크 구축
- 재사용 가능한 작은 단위의 지식

---

### 📚 Resources (자료)
**참고용 자료와 도구**

나중에 다시 볼 수 있는 유용한 정보들입니다.

**예시:**
- **개발 팁**: Git 명령어 치트시트, SQL 최적화 기법
- **도구**: VS Code 확장, 생산성 앱 추천
- **학습 자료**: Python 강의 노트, 경제 용어 정리

---

### 🗄️ Archives (아카이브)
**완료되거나 비활성화된 내용**

더 이상 진행되지 않지만 참고 가치가 있는 내용들입니다.

## 💡 Why PARA + Zettelkasten?

**PARA 방법론:**
- 명확한 구조로 정보 분류
- 프로젝트 중심 사고 촉진
- 생산성 향상

**제텔카스텐:**
- 지식의 재사용성
- 창의적 연결과 통찰
- 살아있는 지식 베이스
