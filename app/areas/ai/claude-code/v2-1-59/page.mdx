## Auto-Memory (자동 메모리)

Claude가 작업하면서 자동으로 유용한 컨텍스트를 메모리에 저장합니다.

### 주요 기능
- 자동 메모리 기록 및 회상
- 세션 간 사용자 선호도 유지
- 프로젝트 컨텍스트 자동 저장

### 관리 방법
```bash
/memory
```

### 활용 방법
- 프로젝트 선호도를 한 번 설정하면 자동 기억
- 반복 작업 시 컨텍스트 재입력 불필요
- 장기 프로젝트에서 일관성 유지

## "Summarize from here" 기능

메시지 선택기에 새로운 옵션이 추가되었습니다.

### 사용법
1. 메시지 선택기 열기
2. "Summarize from here" 클릭
3. 선택한 지점부터의 대화 요약

### 장점
- 부분 대화 요약 가능
- 긴 대화 세션의 효율적인 관리
- 중요한 컨텍스트 빠른 파악
