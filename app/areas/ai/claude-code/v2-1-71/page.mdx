## /loop 명령어

반복 간격으로 프롬프트 또는 슬래시 명령을 실행할 수 있는 새로운 명령어입니다.

### 사용법
```bash
/loop 5m "npm test"
```

### 주요 기능
- 정기적인 작업 자동화
- 세션 내 반복 작업 설정
- 시간 간격 지정 가능

### 활용 예시
- 정기적인 테스트 실행 자동화
- 빌드 상태 체크 자동화
- CI/CD 워크플로우 통합

## 음성 인식 확장

10개 언어가 추가되어 총 20개 언어를 지원합니다.

### 새로 추가된 언어
- 러시아어
- 폴란드어
- 터키어
- 네덜란드어
- 우크라이나어
- 그 외 5개 언어

### 개선 사항
- 저장소 이름 인식 개선
- 개발 용어 정확도 향상 (regex, OAuth, JSON 등)
- 개발자 워크플로우에 최적화
