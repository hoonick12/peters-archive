# Claude Code 2026년 신기능

**2026년에 추가된 주요 기능 정리**

---

## 최신 릴리스 (v2.1.76 - 2026.03.14)

### MCP Elicitation Support
- MCP 서버가 작업 중간에 구조화된 입력을 요청할 수 있는 대화형 기능
- 새로운 Elicitation 및 ElicitationResult 훅 추가
- 더 스마트한 대화형 워크플로우 지원

### 세션 관리 개선
- `-n / --name` CLI 플래그 추가
- 세션 시작 시 표시 이름 설정 가능
- 여러 세션 관리가 더 편리해짐

### 대형 모노레포 지원
- `worktree.sparsePaths` 설정 추가
- git sparse-checkout을 통해 필요한 디렉토리만 체크아웃
- 대규모 프로젝트에서 성능 향상

---

## Memory & Intelligence

### 자동 메모리 기능
- Claude가 작업하면서 자동으로 메모리 기록 및 회상
- 세션 간 사용자 선호도 및 프로젝트 컨텍스트 유지
- 반복 작업 시 더 스마트한 응답

### 대화 요약
- "Summarize from here" 옵션 추가
- 메시지 선택기에서 부분 대화 요약 가능
- 긴 대화 세션의 효율적인 관리

---

## 자동화 & 스케줄링

### /loop 명령어
- 반복 간격으로 프롬프트 또는 슬래시 명령 실행
- 정기적인 작업 자동화
- 세션 내 반복 작업 설정

### Cron 스케줄링
- 세션 내에서 반복 프롬프트 예약 가능
- `CLAUDE_CODE_DISABLE_CRON` 환경 변수로 즉시 중지
- 백그라운드 작업 자동화

---

## 음성 & 인터페이스

### 음성 인식 확장
- 10개 언어 추가 (총 20개 언어 지원)
- 러시아어, 폴란드어, 터키어, 네덜란드어, 우크라이나어 등
- 더 넓은 글로벌 사용자 지원

### 음성 입력 정확도 향상
- 저장소 이름 인식 개선
- 개발 용어 정확도 향상 (regex, OAuth, JSON 등)
- 개발자 워크플로우에 최적화

---

## 개발자 경험

### /effort 명령어
- 모델 노력 레벨 설정 가능
- 작업 복잡도에 따른 세밀한 제어
- 성능과 품질 간 균형 조절

### Agent Teams (Research Preview)
- 멀티 에이전트 협업 기능
- 복잡한 작업을 여러 에이전트가 분담
- 병렬 처리로 효율성 향상

### /claude-api 스킬
- Claude API 및 Anthropic SDK로 애플리케이션 구축
- API 통합 개발 지원
- SDK 사용 가이드 및 예제

### Bash 자동 승인 확장
- lsof, pgrep, tput, ss, fd, fdfind 명령어 추가
- 권한 프롬프트 감소
- 더 부드러운 개발 워크플로우

## 활용 팁

### Memory 기능 활용
- 프로젝트 선호도를 한 번 설정하면 자동 기억
- 반복 작업 시 컨텍스트 재입력 불필요
- 장기 프로젝트에서 일관성 유지

### /loop를 이용한 모니터링
- 정기적인 테스트 실행 자동화
- 빌드 상태 체크 자동화
- CI/CD 워크플로우 통합

### Agent Teams 활용
- 프론트엔드/백엔드 동시 개발
- 테스트와 구현 병렬 진행
- 복잡한 리팩토링 작업 분산
