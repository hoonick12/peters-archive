## MCP Elicitation Support

MCP 서버가 작업 중간에 구조화된 입력을 요청할 수 있는 대화형 기능이 추가되었습니다.

### 주요 기능
- 새로운 Elicitation 및 ElicitationResult 훅 추가
- MCP 서버가 워크플로우 중간에 사용자 입력 요청 가능
- 더 스마트한 대화형 워크플로우 지원

### 사용 예시
MCP 서버가 API 키, 설정 값, 또는 사용자 선택이 필요할 때 실시간으로 요청하고 응답받을 수 있습니다.

## 세션 관리 개선

### `-n / --name` CLI 플래그
```bash
claude-code -n "프로젝트A 작업"
```

- 세션 시작 시 표시 이름 설정 가능
- 여러 세션을 동시에 관리할 때 구분이 쉬워짐
- 세션 전환 시 작업 컨텍스트 파악 용이

## 대형 모노레포 지원

### `worktree.sparsePaths` 설정

대규모 모노레포에서 필요한 디렉토리만 체크아웃하여 성능을 향상시킬 수 있습니다.

```json
{
  "worktree": {
    "sparsePaths": [
      "packages/frontend",
      "packages/api"
    ]
  }
}
```

### 장점
- git sparse-checkout 자동 설정
- 불필요한 파일 로드 방지
- 대규모 프로젝트에서 빠른 시작 시간
