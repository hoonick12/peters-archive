## Agent Teams (Research Preview)

멀티 에이전트 협업 기능이 연구 프리뷰로 추가되었습니다.

### 주요 기능
- 복잡한 작업을 여러 에이전트가 분담
- 병렬 처리로 효율성 향상
- 각 에이전트가 전문 영역에 집중

### 활용 사례
- 프론트엔드/백엔드 동시 개발
- 테스트와 구현 병렬 진행
- 복잡한 리팩토링 작업 분산

### 주의 사항
- Research Preview 기능으로 실험적 단계
- 프로덕션 환경에서는 신중하게 사용
